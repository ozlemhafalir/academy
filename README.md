# Academy

This is the UI for the Academy Project.


## Features
- Browsing categories and instructors
- Applying for a course
- Registering as a student
- Registering as an instructor
- Managing courses
- Managing categories

## Installation
```shell
$ git clone
$ cd academy
$ npm install
$ npm start
```

## Deployment
This app runs on Heroku
